## ⚙️ Como Usar?
1. Clone este repositório: 

```bash
git clone https://github.com/RaizaCirne/rolagem-fotografica.git
```
<br />

2. Navegue para o projeto desejado e instale as dependências (se aplicável):
```bash
cd nome-do-projeto
npm install
```
<br />

3. Execute o projeto: 
```bash
npm start
```
